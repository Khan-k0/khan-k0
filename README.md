# 🙋 About Me

안녕하세요. 꾸준한 학습과 성장을 즐기는 개발자 지망생입니다.

새로운 기술을 배우는 것에 그치지 않고, 이를 직접 프로젝트에 적용하며 문제를 해결하는 
과정을 좋아합니다. 작은 기능 하나라도 직접 구현하고 개선해 나가며 경험을 쌓고 있습니다.

심리학을 전공한 경험을 바탕으로 기술뿐만 아니라 **사용자의 입장에서 생각하는 개발**을 중요하게 여기고 있습니다. 
사용성과 편의성을 고려한 서비스를 만들기 위해 끊임없이 배우고 도전하고 있습니다.

현재는 Java, Spring Boot, SQL을 중심으로 백엔드 개발을 공부하고 있으며, 다양한 프로젝트를 통해 실무에 
가까운 경험을 쌓아가고 있습니다.

"오늘보다 더 나은 개발자"가 되기 위해 꾸준히 성장하고 있습니다.
